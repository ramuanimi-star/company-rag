\# Sample Documents



This folder contains \*\*non-confidential, sample PDF documents\*\* used only for

demonstration and testing of the HR RAG system.



❌ No company, client, or sensitive documents are included here.  

✅ You may replace these files with your own local documents for testing,

but do \*\*not\*\* commit them to Git.



For real usage, place documents in a local-only folder such as `data/`

(which is intentionally ignored by Git).
